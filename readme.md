****

## **Setup☕**
> 1- `npm i aoi.js@latest`

> 2- `npm i @akarui/aoi.music@latest`

> 3- `npm i aoi.db`

> 4- `npm i express`

> 5- `npm i ffmpeg-static@latest`

> 6- `Enable Intents`
![Enable Intents](https://cdn.wild-life-studio.ir/assets/image/discord-intents.png) 
****
